概述

贵州财经大学《关于2016年度贵州财经大学在校学生科研项目申报的通知》已经发布，现拟定3个题目， 欢迎各年级有组织能力，有科研创新能力的同学组织团队以此三项题目之一申报上述项目。在对项目内容有一定理解，并有决心做好项目，可以以团队方式撰写项目书，联系丁红发做指导老师。

候选题目及内容

  • 题目一：大数据交易及数据服务中的隐私问题研究

    1. 梳理国内数据交易相关政策中的隐私条款
    2. 梳理国内关于数据交易和数据服务的专家观点
    3. 分析大数据交易及数据服务可能面临的问题
    4. 对大数据及数据服务中隐私问题的对策

  • 题目二：国内大数据交易及数据服务平台对比研究

    1. 全面梳理调研国内大数据交易平台及数据服务平台
    2. 大数据交易及数据服务平台的特征分析
    3. 大数据交易及数据服务平台的对比
    4. 对贵州大数据交易及数据服务的建议

  • 题目三：贵州省数据开放共享的政策分析研究

    1. 梳理贵州省数据开放共享政策
    2. 梳理国内发达省份数据开放共享的政策
    3. 国内大数据开放共享政策需求分析
    4. 对贵州大数据开放共享政策的意见

联系方式

邮箱： gufe.hfding@foxmail.com

时间安排

  1. 2016年10月20日22:00之前写好项目申报书联系丁老师，发到丁老师邮箱，丁老师针对各项目按照题目归类优选后确定队伍，至多3队。
  2. 2016年10月23日22:00之前修改第一稿
  3. 2016年10月25日22:00 之前修改第二稿
  4. 2016年10月27日22:00之前定稿，并发到学院

双方权利义务

团队责任

  1. 团队全员积极筹划，按照校方要求撰写申报书，并按时提交校方；
  2. 申报成功后，严格按照校方要求安排好研究进度；
  3. 积极以个人或团队形式发表论文、研究报告、专利，并积极往高层次刊物会议发表。若发表高层次成果，指导教师提供一定奖金；
  4. 队长按照校方财务要求将项目经费用以团队项目研究；
  5. 虚心接受导师指导，并每周定期以书面或口头形式向导师汇报研究进度；
  6. 研究素材和研究成果，在项目实施过程和结题时移交给导师；
  7. 保证项目能够按时结题；
  8. 可以项目成果申报院、校等各级奖励。

导师责任

  1. 指导项目团队撰写申报书；
  2. 督促团队展开研究、撰写研究报告、论文、专利，并定期检查，直至按时结题；
  3. 若团队成员发表高层次成果，提供成果作者一定奖金（中文核心、CSSCI或EI会议论文，1000；EI期刊、SCI期刊以上1500元）；
  4. 二次使用团队研究成果，尊重成果作者知识产权。
