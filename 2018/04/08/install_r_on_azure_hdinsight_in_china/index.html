参考MSDN链接为，该链接适用Global Azure。

https://azure.microsoft.com/en-us/documentation/articles/hdinsight-hadoop-r-scripts/

此过程得到了世纪互联的工程师Zhang Xiangyu，Jun Chen，He Shu的帮助。
