How about the Application of verifiable shuffle fully homomorphic schemes in Genome data privacy preserving

Recently, a verifiable shuffle version of full homomorphic encryption scheme is posted on eprint.iacr.org.

Cryptology ePrint Archive: Report 2018/027

A verifiable shuffle for the GSW cryptosystem

Martin Strand

Shuffles are useful in e-voting, traffic routing and other applications. And verifiable shuffle provides some new specialities for these application.

As Full Homomorphic Encryption is very useful in Genomic data privacy preserving, some intresting topics should be disscussed further with this new result.

  • what’s the meanning of verifiable shuffle?
  • what’s the gap in genomic data privacy preserving?
  • what’s the difficulties to apply this new scheme?
  • is it possible to design and develop a new system to protect the privay of genomic data?
