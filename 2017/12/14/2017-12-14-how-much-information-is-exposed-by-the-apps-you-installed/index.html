How much information is exposed by the apps you installed ?

Some academic work and techonical work showed that, apps list can be got by a third party app via the api canopenURL and othe ones.

A very early app named Dump is developed to recommand new app to users. this app indecated that it can get all the installed apps over the users mobile phone. Later, the develop team of this app joined into google.

And in the last year, for privacy issue, apple limited the use of CanOpenURL api on IOS 9 and later. You can see the explanation here.

What I want to say is that, over android, this api is still useable. For the commencial puspose, i.e advertising, this API and other related apis will be used to track and analysis users.
